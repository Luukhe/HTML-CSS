ANOTAÇÕES: Nesta aula, vimos as novas pseudo classes:

  • visited: Permite trocar a cor de um link JÁ VISITADO
  • active: Permite o efeito da troca de cor de um CLIQUE ATIVO, de quando estamos com o click segurado, haver uma TROCA DE COR.

VIMOS TAMBÉM, QUE ERA O FOCO, OS PSEUDO ELEMENTOS. Os pseudo elementos são representados por DOIS PONTOS, DUAS VEZES: "::". Os pseudo elementos podem ser usados tanto nas classes quanto nos elementos (assim como as pseudo classes também...). Pelo que eu entendi, os pseudo elementos tem praticamente a mesma função das pseudo classes... (MAS EU ACHO QUE O INTUITO É MEXER MAIS COM CONTEÚDO)

Exemplos vistos na aula:

  • after: Permite ADICIONAR coisas (JUNTAMENTE COM O COMANDO CONTENT: ' ') DEPOIS, como o nome sugere, do elemento escolhido, ao invés de ser adicionado diretamente na tag/link do elemento.
  • before: Funciona da mesma forma que o after (JUNTAMENTE COM O COMANDO CONTENT: ' '), mas, como o nome sugere, o que for adicionado, é adicionado ANTES.
  • *As informações adicionais deverão ser colocadas dentro das aspas do content*
