Inserindo vídeos hospedados localmente

Este vídeo está hospedado no meu próprio servidor!

Anotações minhas: da mesma forma que os áudios, os vídeos também existem em diferentes formatos e também podem ser carregados no site com os diferentes formatos, a fim de agradar os diferentes navegadores que possam ou não sem compatíveis com algum formato de vídeo colocado.

E também, da mesma forma que os áudios, existe a forma mais simples que é com a tag "VIDEO", acompanhado do src. Ou seja, a forma de carregar múltiplos vídeos de diferentes formatos é a MESMA dos áudios (remove-se o parâmetro SRC da TAG VIDEO, e coloca DENTRO da tag, uma OUTRA TAG: "SOURCE", como visto na aula sobre upload dos áudios).
OBS: o vídeo quando carregado primeiramente, ocupará, na TELA, o tamanho dele em PIXELS! Ou seja, um vídeo de 720p, ocupará 720 pixels da tela. Usa-se o parâmetro WIDTH para diminuí-lo.
OBS 2: Da mesma forma que os áudios, quando o vídeo carregado SEM o parâmetro CONTROLS, ficará só uma imagem estática na tela. Por isso, usa-se TAMBÉM o parâmetro CONTROLS!
OBS 3: O parâmetro PRELOAD com os valores METADATA, NONE E AUTO também são utilizados!

Código para colocar o vídeo de forma simplificada:
<video src="midia/video.mp4" width="220" controls preload="metadata"></video>

LEMBRANDO: quando formos colocar vários vídeos de formatos diferentes a serem carregados, a ORDEM deles importará para QUAL será carregado primeiro!
Código para colocar o vídeo em diferentes formatos:

        
            <video width="300" poster="imagens/limoes-capa.png" preload="metadata" controls>
                <source src="midia/video.mp4" type="video/mp4">
                <source src="midia/video.m4v" type="video/m4v">
                <source src="midia/video.webm" type="video/webm">
                <source src="midia/video.ogv" type="video/ogg">
                <p>Seu navegador não tem compatibilidade com reprodução de vídeo <a href="midia/video.mp4">Clique aqui para baixar o arquivo MP4</a></p>
            </video>

PARÂMETRO MUITO INTERESSANTE PARA VÍDEOS: o parâmetro "POSTER" permite colocarmos uma CAPA para o vídeo (logicamente, ele vai pedir a imagem que é pra colocar), para que ele comece com uma imagem de escolha nossa ao invés de começar com a imagem inicial do vídeo!

NOTA:esse método consome MUITA banda, sairá caro para o cliente!

Inserindo vídeos do YouTube

Para inserir vídeos do YT, basta ir no vídeo em si, clicar em COMPARTILHAR, e depois INCORPORAR. Um link será gerado pelo próprio YT, então, é só clicar em copiar.

VANTAGEM:A grande vantagem de usar um vídeo que está carregado em OUTRA plataforma, é que quando é dado PLAY no vídeo, os dados consumidos pelo vídeo são consumidos PELA PLATAFORMA EXTERNA, e não pelo seu próprio host.

Já que o YouTube em si é também uma ferramenta do Google e é a SEGUNDA maior plataforma de busca, é ALTAMENTE RECOMENDADO que o cliente tenha um canal no YT e que os vídeos do site dele sejam linkados do próprio YT pois, se o vídeo do cliente está NO SITE e além do site, está no YT também, isso AMPLIA AINDA MAIS a visualização, a abrangência que o algorítmo do google dará, já que o YouTube TAMBÉM é do Google.
