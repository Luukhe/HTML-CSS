Em resumo sobre o CSS:

  • Estilo INLINE: Style inline, como o nome já sugere, é a forma de usar CSS DIRETAMENTE na tag HTML, o qual até funciona, mas NÃO é o melhor modo, desde que sendo um site ou um projeto grande, teria que ser feito de linha em linha e sendo assim, não é um método nada OTIMIZADO e, PRINCIPALMENTE, é um método DESORGANIZADO, pois o style se misturaria todo com as tags e ficaria muita poluição no código.

  • Estilos Locais ou Internos: É uma forma mais organizada de fazer o estilo de uma página, pois a parte do estilo (a tag style) fica separada do código, na parte do HEAD, não poluindo assim o código HTML da página. A desvantagem é que, para um projeto com MUITAS PÁGINAS, o estilo teria que ser colocaod em CADA UMA delas, o que tornaria difícil a manutenção, caso necessária.

  • Estilos Externos (style.css): A melhor forma e a mais indicada para se usar CSS, pois além de ficar em um arquivo separado para ficar mais organizado, pode ser feito e LINKADO para TODAS as páginas.

DICA LEGAL:
No ARQUIVO do CSS, cria-se uma REGRA para podermos usar ACENTUAÇÕES e/ou SIMBOLOGIAS no CSS, assim: @CHARSET "UTF-8"; (basicamente, do mesmo jeito que fica no HEAD do HTML, só que dessa forma ai, com o arroba, pois é no CSS...)
