Imagem dinâmica

Tente abrir este site em vários dispositivos diferentes ou simplesmente aumente e diminua o tamanho do seu navegador.

A tag PICTURE, é uma tag utilizada para carregar imagens com DIFERENTES tamanhos, para diferentes PIXELS de TELA.

Código utilizado:

        
            <picture>
                <source media="(max-width: 720px)" srcset="imagens/foto-p.png" type="image/png">
                <source media="(max-width: 1050px)" srcset="imagens/foto-m.png" type="image/png">
                <img src="imagens/foto-g.png" alt="Imagem flexível">
            </picture>

Reproduzindo Áudio

Vamos aprender a reproduzir áudios em HTML5.

Observações minhas: a tag mais simples usada para reproduzir um áudio em um site, é a tag <audio>. Mas, ela sozinha não funcionará (pelo que eu vi), é necessário, também, adicionar os parâmetros "CONTROLS" e "AUTOPLAY". O parâmetro controls, como o nome propriamente diz, te permite usar um mini player de música, onde (obviamente) você pode mutar o som, diminuir volume e dar play/pause.
OBS: TESTEI SÓ COM O PARÂMETRO CONTROLS E FUNCIONOU NORMALMENTE!!! FUNCIONOU SEM O AUTOPLAY!!

O código ficará assim:
<audio src="midia/retribution-neffex.mp3" controls></audio>

Outra forma de reproduzir áudio, será usando novamente a tag AUDIO, PORÉM, RETIRA-SE O PARÂMETRO SRC e adiciona-se, DENTRO da tag AUDIO, a --TAG-- "SOURCE:SRC", ELA SIM COM O PARÂMETRO SRC!! Sim, a mesma usada para fazer imagens dinâmicas acima. Será adicionado, também, o parâmetro "TYPE" junto, onde será preciso especificar o FORMATO do áudio. A grande VANTAGEM de usar esse modo, é que podemos colocar (dentro da tag audio) vários sources com o mesmo arquivo, só que com DIFERENTES FORMATOS, para caso de algum navegador NÃO conseguir reproduzir algum formato. E, LEMBRANDO: a ORDEM de cada source importa, será a ordem de QUAL formato será carregado primeiro.

Exemplo de como ficaria e com os TYPES especificados de cada formato:

        
            <audio controls>
                <source src="midia/retribution-neffex.mp3" type="audio/mpeg">
                <source src="midia/retribution-neffex.ogg" type="audio/ogg">
                <source src="midia/retribution-neffex.wav" type="audio/wav">
                <p>Infelizmente seu navegador não consegue reproduzir áudio. <a href="midia/retribution-neffex.mp3">Clique aqui para baixar o arquivo MP3.</a></p>
            </audio>

Bonus: Se caso nenhum dos formatos funcionar no navegador da pessoa, um link para baixar o áudio.

Existe também o parâmetro "preload" para a tag audio. O parâmetro preload tem três valores: auto, metadata e none.

  1. Auto: só vai ser considerado que o site foi completamente carregado DEPOIS de carregar também o ÁUDIO junto. Ou seja, ele carrega o áudio juntamente do site, podendo dificultar a navegação do usuário.
  2. Metadata: carregará só o básico do arquivo (exemplo: duração, nome do arquivo, data etc) e só carregará o áudio em si quando o usuário der play.
  3. None: como o nome sugere, não carregará nada antes do usuário dar PLAY.
