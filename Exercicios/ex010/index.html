PS: É sempre bom usar o parâmetro TARGET="_BLANK", que redireciona o site do link para uma nova aba sem sair do seu site. (O target blank é um parâmetro que pode ficar tanto antes quanto depois do link). Já o parâmetro REL (com o valor EXTERNAL), indica para os mecanismos de busca que o link é um link externo ao seu site.

Usando links INTERNOS

Os links internos é um link que vai fazer a RELAÇÃO com outra página que está DENTRO do nosso próprio servidor, que condiz e diz respeito à nossa própria página. Porém, antes de um link interno ser criado, precisamos ter OUTRAS PÁGINAS (dentro do site).

Esta é a primeira página do site. Se você quiser, pode acessar também a minha segunda página

PS: Utiliza-se também, parâmetros REL para links INTERNOS, com o valor "next" para uma página a ser levada, e o valor "prev" para uma página que tem um link interno para VOLTAR para a primeira. Isso facilica os mecanismos de busca a encontrarem como é o sistema de navegação que você está criando.

Em suma, para criar um link externo, usa-se a mesma tag (tag: "A", de ANCHOR) usada para criar links externos. A única diferença, é que para um link interno é necessário colocar o CAMINHO da nova página.

Outro VALOR bem importante para o parâmetro REL, é o valor "nofollow". Esse valor indica que o link utilizado NÃO é sabido que seja um link OFICIAL, o link não tem o seu AVAL, quando você não quer que o mecanismo de busca perpetue essa busca.

Links para download

Também usa-se a tag "A", de anchor. Podemos, inclusive, mesclar as coisas que aprendemos até agora, como simplesmente colocar os links para download em listas! É interessante, também, enquanto estivermos só criando as tags dos links e não tivermos um link ainda para colocar, usar uma HASHTAG (#) no lugar do link, para indicar que é um link vazio.

ADENDO: No vídeo da aula, o professor também usou os parâmetros "download" e "type" para fazer os downloads dos arquivos. Não funcionou com o arquivo em pdf, pois o navegador já reconhece o pdf por si só (não entendi muito). Então, nesse caso, é mais fácil colocar um target blank, pois em arquivos pdf sempre tem o botãozinho de imprimir ou salvar. (NOTA: para utilizar o parâmetro TYPE, é preciso colocar valores referentes aos respectivos FORMATOS dos arquivos, por exemplo: type="application/pdf".)

ADENDO 2: No caso do arquivo ZIPADO, o professor também utilizou os mesmos parâmetros, só mudou (obviamente) o parâmetro do TYPE, usando: type="application/zip". Porém, tanto com quanto sem os "novos" parâmetros e valores, o download no caso do ZIP funcionou normalmente.
