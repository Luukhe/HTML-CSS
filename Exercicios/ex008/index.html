Principais formatações

Negrito / Destaque

Nessa frase, temos um termo em negrito usando a tag B (que é uma forma NÃO SEMÂNTICA) de se usar o negrito.

Já nesta frase, temos um termo em DESTAQUE usando a tag STRONG (que é uma tag SEMÂNTICA).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I (não semantica)

Nesta frase, temos um termo em ênfase usando a tag EM (SEMÂNTICO)

Texto marcado

Podemos criar também um texto marcado usando a tag MARK. (Por padrão, a tag MARK já vem com aquela cor de "marca-texto", podemos alterá-la usando um parâmetro chamado STYLE, que é melhor usado quando for para mexer em uma tag por si só".

E no outro parágrafo, temos outro texto marcado no final.

Texto grande e Pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo. (Podemos ver que a tag BIG entrou/vai entrar em desuso por ser só uma tag de FORMA que podemos facilmente aumentar no css. Já a tag SMALL ainda existe por ainda existir alguma SEMÂNTICA para ela)

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido mas não considerado. (Esse é o efeito de um texto RASURADO, riscado.)

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. COMPLEMENTO: a tag INS dá o efeito de SUBLINHADO. Existe também a tag U que possui o mesmo efeito, porém é uma tag NÃO SEMÂNTICA. Já o INS, É SEMÂNTICA
